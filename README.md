# 合同管理系统简介 java1201

本项目是一款基于 Springboot + Vue 技术栈开发的合同管理系统，采用前后端不分离的模式，涵盖了合同管理的全生命周期，确保了系统的高效性和易用性。

## 项目概述

系统主要分为用户和管理员两个角色，各自拥有不同的功能权限，以保障系统操作的安全性和便捷性。

## 技术栈

- 后端：Spring、SpringMVC、MyBatis
- 前端：Vue
- 运行环境：JDK1.8、IDEA、MySQL、Tomcat8

## 功能模块

### 管理员

- **用户管理**：负责用户注册、登录、权限设置等，确保系统用户的安全性和合规性。
- **基础数据管理**：维护部门、职位、角色等基础数据，确保数据的准确性和一致性。
- **公告信息管理**：发布合同管理相关的通知、公告和更新信息，促进信息共享和传播。
- **留言管理**：查看并处理用户留言，解答问题、收集反馈。
- **轮播图信息管理**：设置和更新前台门户的轮播图，展示重要的合同信息、公告或宣传内容。
- **合同管理**：负责合同的起草、审批、签订、执行、归档等全流程管理。
- **客户管理**：记录和管理与合同相关的客户信息，包括客户基本资料和历史合同记录。

### 用户

- **前台门户浏览**：通过前台门户了解系统功能和操作。
- **个人信息管理**：查看和编辑个人基本资料。
- **合同查看与操作**：根据权限查看相关合同信息，并执行如签署、查看进度等操作。
- **留言功能**：在系统中留言，提出疑问、建议或反馈。
- **公告信息查看**：了解合同管理相关的通知和更新。

## 系统角色

- 管理员：负责系统的日常管理和维护。
- 用户：使用系统进行合同相关的操作和浏览。
